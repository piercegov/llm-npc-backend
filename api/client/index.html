    Client API Reference¶

    The client module provides the main interface for interacting with the LLM NPC Backend.

    NPCClient¶

    NPCClient ¶

    NPCClient(base_url: str = 'http://localhost:8080')

    Main client for interacting with the LLM NPC Backend.

    Usage

    client = NPCClient("http://localhost:8080")

    with client.session("my-game-session") as session: npc = session.create_npc("Gandalf", "A wise wizard") response = npc.act(surroundings=["Dark cave", "Dragon"])

    Initialize the NPC client.

    Parameters:

    Name Type Description Default
    base_url str

    Base URL of the backend server

    		 'http://localhost:8080'

    health_check ¶

    health_check() -> bool

    Check if the backend is running.

    Returns:

    Type Description
    bool

    True if backend is healthy, False otherwise

    session(session_id: str) -> Session

    Name Type Description Default

    Unique identifier for this game session

    		 required

    Type Description

    A Session instance

    list_npcs() -> Dict[str, Any]

    Initialize a session.

    Parameters:

    Name Type Description Default

    The NPCClient instance

    		 required
    session_id str

    Unique identifier for this session

    		 required

    NPC¶

    NPC ¶

    NPC(client: NPCClient, session: Session, npc_id: str, name: str, background: str)

    Represents an NPC that can perform actions.

    Usage

    response = npc.act( surroundings=["Forest", "Sword on ground"], events=["You found a weapon"] )

    Initialize an NPC instance.

    Parameters:

    Name Type Description Default
    client NPCClient

    The NPCClient instance

    		 required
    session Session

    The Session this NPC belongs to

    		 required
    npc_id str

    Backend NPC ID

    		 required
    name str

    NPC name

    		 required
    background str

    NPC background story

    		 required

    act ¶

    act(surroundings: Union[List[str], List[Dict[str, str]], List[Surrounding], Surroundings, ContextBuilder], events: Optional[Union[List[str], List[Dict[str, str]], List[Event]]] = None, knowledge_graph: Optional[Union[KnowledgeGraph, Dict]] = None) -> Response

    Execute an action/tick for this NPC.

    Parameters:

    Name Type Description Default
    surroundings Union[List[str], List[Dict[str, str]], List[Surrounding], Surroundings, ContextBuilder]

    What the NPC can see/interact with. Can be: - Simple list of strings (converted to surroundings) - List of dicts with "name" and "description" - List of Surrounding objects - Surroundings builder - ContextBuilder (other args ignored if used)

    		 required
    events Optional[Union[List[str], List[Dict[str, str]], List[Event]]]

    Recent events that occurred. Can be: - Simple list of strings - List of dicts with "event_type" and "event_description" - List of Event objects

    		 None
    knowledge_graph Optional[Union[KnowledgeGraph, Dict]]

    Optional knowledge graph for NPC memory. Can be: - KnowledgeGraph object - Dict with "nodes" and "edges"

    		 None

    Returns:

    Type Description
    Response

    Response object with NPC's action result

    Raises:

    Type Description
    BackendError

    If the action fails

    Usage Example¶

    from llm_npc import NPCClient, tool

@tool
def speak(message: str):
    """Make the NPC speak"""
    print(f"NPC says: {message}")

# Create client
client = NPCClient("http://localhost:8080")

# Check health
if client.health_check():
    print("Backend is running")

# Create session
with client.session("game-123") as session:
    # Register tools
    session.register_tools([speak])

    # Create NPC
    wizard = session.create_npc(
        name="Merlin",
        background="A wise old wizard"
    )

    # Make NPC act
    response = wizard.act(
        surroundings=["Tower room", "Ancient books"],
        events=["A visitor arrives"]
    )

    print(response.text)

# List all NPCs
npcs = client.list_npcs()
print(f"Total NPCs: {npcs['count']}")

# Delete an NPC
client.delete_npc(wizard.npc_id)
